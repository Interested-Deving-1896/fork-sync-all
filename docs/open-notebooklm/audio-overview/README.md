# Audio Overview

PDF-to-podcast audio generated by open-notebooklm from fork-sync-all documentation.

Uses Llama 3.3 70B (Fireworks AI) for two-speaker dialogue generation and
MeloTTS or Suno Bark for speech synthesis.

**Variants:**

| Directory | Description |
|---|---|
| [`standard/`](standard/) | Default two-speaker podcast format |

**File format:** `.mp3`
